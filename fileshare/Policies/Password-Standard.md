# Password & Authentication Standard (v2.1)
Effective: 2025-09-15 | Owner: IT Security

- Minimum 14 characters for user accounts; passphrases encouraged.
- Service accounts: 30+ characters, vaulted, rotated on personnel change.
- MFA required for: VPN, email, admin consoles, and the AI gateway.
- Legacy protocols (NTLMv1, basic auth) are disabled tenant-wide.
